# ShoppingMall

## 한눈에 보는 통합 온라인 쇼핑몰

### 프로젝트 기간
2023.01 ~ 2023.02(5주)
### 개발 인원
1명(토이 프로젝트)

* * * 

프로젝트 개발환경은 다음과 같습니다.

#### 개발환경
* JSP, Servlet
* Java, MySql
* Apache Tomcat 9.0

#### 구현 부분
* 사용자 페이지
    - 회원가입 및 로그인(회원 정보 수정)
    - 상품 상세 페이지
    - 카테고리 분류(의류, 도서, 식품)
    - 상품 명으로 검색
    - 장바구니
    - 댓글
    - 회원 탈퇴
